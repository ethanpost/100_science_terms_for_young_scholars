# Periodic table

Imagine walking into a huge library where every book has been thrown onto the floor.

There are adventure stories, histories, science books, poems, and mysteries, but they are all mixed together. Finding the book you need would be slow and frustrating.

Now imagine the same library with shelves, labels, sections, and a catalog. Suddenly the books make sense. You can find what you need. You can even guess where a new book belongs.

The periodic table does something like that for chemistry.

**The periodic table is a chart that organizes all known chemical elements by atomic number and by repeating patterns in their properties.**

It is one of the most useful maps in science.

It does not map countries, roads, or rivers.

It maps the building blocks of matter.

## The Elements on the Table

An **element** is a pure substance made of only one kind of atom.

Gold is an element.

Oxygen is an element.

Carbon is an element.

Iron is an element.

Hydrogen is an element.

Every element has its own atoms. A carbon atom is not an oxygen atom. An iron atom is not a gold atom.

The periodic table lists the elements in an organized way so scientists can compare them, predict their behavior, and use them wisely.

## The Periodic Table Is Not Just a List

At first glance, the periodic table may look like a strange wall of boxes.

Each box usually contains information such as:

- The element's name
- The element's chemical symbol
- The element's atomic number
- The element's atomic mass

But the table is more than a list of facts.

Its shape matters.

Where an element sits on the table tells you something about how that element behaves.

That is why chemists treat the periodic table almost like a map, a dictionary, and a puzzle key all at once.

## Reading One Box

Look at carbon as an example.

Carbon's chemical symbol is **C**.

Its atomic number is **6**.

Its name is **carbon**.

Its atomic mass is about **12**.

The exact layout depends on the periodic table you are using, but most element boxes give you these same basic clues.

Once you learn how to read one box, you can begin to read the whole table.

## Chemical Symbols

A **chemical symbol** is a short way to write the name of an element.

Some symbols use one capital letter:

- H for hydrogen
- C for carbon
- N for nitrogen
- O for oxygen
- S for sulfur

Some symbols use two letters:

- He for helium
- Li for lithium
- Be for beryllium
- Cl for chlorine
- Ca for calcium

The first letter is always capitalized. The second letter, if there is one, is lowercase.

This matters.

**Co** means cobalt.

**CO** means carbon monoxide, a compound made from carbon and oxygen.

One tiny capital letter can change the meaning completely.

## Atomic Number

The **atomic number** is the number of protons in the nucleus of an atom.

A **proton** is a positively charged particle in the center of an atom.

Atomic number identifies the element.

Every hydrogen atom has 1 proton.

Every helium atom has 2 protons.

Every carbon atom has 6 protons.

Every oxygen atom has 8 protons.

Every iron atom has 26 protons.

If an atom has 8 protons, it is oxygen. If it has 9 protons, it is fluorine. If it has 10 protons, it is neon.

Change the number of protons, and you change the element.

## The Order of the Table

The modern periodic table is arranged mostly by atomic number.

Hydrogen, atomic number 1, comes first.

Helium, atomic number 2, comes next.

Lithium, atomic number 3, follows.

The order continues across the table.

This order is not random. It follows the number of protons in each atom.

Because atoms with certain proton numbers also have certain electron arrangements, patterns appear again and again.

Those repeating patterns are the reason the table is called **periodic**.

## What Periodic Means

The word **periodic** means happening again in a pattern.

The days of the week are periodic. Monday, Tuesday, Wednesday, and the rest repeat in order.

Seasons are periodic. Spring, summer, autumn, and winter come around again.

In the periodic table, chemical properties repeat in a pattern as atomic number increases.

Elements in the same vertical column often behave in similar ways.

This repeating pattern is the great secret of the table.

## Dmitri Mendeleev and the Missing Pieces

In the 1800s, scientists knew about many elements, but they did not yet have the modern table.

Dmitri Mendeleev, a Russian chemist, arranged the elements known in his time by their masses and properties.

He noticed patterns.

Even more impressively, he left empty spaces where he thought undiscovered elements should go.

He predicted some of their properties before those elements were found.

Later discoveries showed that many of his predictions were remarkably close.

Mendeleev's work helped show that the elements are not a messy pile. They follow an order.

## Rows Are Periods

The horizontal rows of the periodic table are called **periods**.

The first period has hydrogen and helium.

The second period begins with lithium and ends with neon.

The third period begins with sodium and ends with argon.

As you move from left to right across a period, atomic number increases by one each step.

That means each element has one more proton than the element before it.

Periods also show changes in properties. Elements on the left side of a period are usually more metallic. Elements on the right side are usually more nonmetallic.

## Columns Are Groups

The vertical columns of the periodic table are called **groups** or **families**.

Elements in the same group often have similar chemical behavior.

This is because their atoms have similar outer electron arrangements.

Electrons are negatively charged particles around the nucleus.

The outer electrons are especially important because they are involved when atoms bond or react.

If elements have similar outer electrons, they often react in similar ways.

That is why the columns of the table are so powerful.

## Group 1: The Alkali Metals

The first column, not counting hydrogen in many classroom discussions, contains the **alkali metals**.

These include lithium, sodium, potassium, rubidium, cesium, and francium.

Alkali metals are very reactive.

They react strongly with water.

Sodium and potassium are important in living things, but pure sodium metal and pure potassium metal must be handled very carefully.

This is a useful lesson in chemistry:

An element can behave very differently depending on its form and what it is combined with.

Sodium in table salt is not the same as pure sodium metal.

## Group 2: The Alkaline Earth Metals

The second column contains the **alkaline earth metals**.

These include magnesium, calcium, strontium, and barium.

They are reactive metals, but generally less reactive than the alkali metals.

Calcium is important in bones and teeth.

Magnesium is used in some metals, fireworks, and living cells.

The group pattern helps you see why elements in the same column can have related uses and behaviors.

## The Transition Metals

The large middle section of the periodic table contains many **transition metals**.

These include iron, copper, nickel, silver, gold, zinc, and many others.

Transition metals are often strong, shiny, and good conductors of heat and electricity.

Many are useful in tools, buildings, wires, coins, machines, and electronics.

Iron helps make steel.

Copper carries electric current in wires.

Gold resists corrosion and is useful in jewelry and electronics.

This middle region of the table contains many of the metals people use every day.

## Metals

Most elements are metals.

Metals are often:

- Shiny when freshly cut or polished
- Good conductors of heat
- Good conductors of electricity
- Malleable, meaning they can be hammered into sheets
- Ductile, meaning they can be drawn into wires
- Solid at room temperature, except for mercury

Metals are found mostly on the left and center of the periodic table.

Metals help build bridges, bicycles, trains, phones, computers, cooking pans, wires, engines, and tools.

## Nonmetals

Nonmetals are found mostly on the right side of the periodic table, plus hydrogen on the far left.

Nonmetals usually do not behave like metals.

Many are poor conductors of heat and electricity.

Some are gases at room temperature, such as oxygen, nitrogen, fluorine, and chlorine.

Some are solids, such as carbon, sulfur, and phosphorus.

Bromine is a liquid at room temperature.

Nonmetals are essential for life.

Oxygen is needed by most animals for respiration.

Carbon is the backbone of living things.

Nitrogen is important in proteins and DNA.

Phosphorus helps form bones, teeth, and genetic material.

## Metalloids

Between the metals and nonmetals is a stair-step region containing **metalloids**.

A metalloid is an element with some properties of metals and some properties of nonmetals.

Common metalloids include boron, silicon, germanium, arsenic, antimony, and tellurium.

Silicon is especially important because it is used in computer chips and solar cells.

Metalloids remind us that nature does not always fit into perfectly sharp categories.

Some elements are in between.

## Group 17: The Halogens

The next-to-last column on the right contains the **halogens**.

These include fluorine, chlorine, bromine, iodine, and astatine.

Halogens are reactive nonmetals.

They often form salts when they react with metals.

Chlorine combines with sodium to form sodium chloride, ordinary table salt.

The halogens can be useful, but many are dangerous in their pure forms.

Chlorine gas, for example, must not be breathed.

Again, chemistry teaches careful thinking:

The properties of a pure element can be very different from the properties of a compound containing that element.

## Group 18: The Noble Gases

The far-right column contains the **noble gases**.

These include helium, neon, argon, krypton, xenon, and radon.

Noble gases are usually very unreactive.

Their atoms have stable outer electron arrangements, so they do not easily form chemical bonds.

Helium is used in balloons because it is lighter than air and does not burn.

Neon can glow in signs when electricity passes through it.

Argon is used in some light bulbs and welding because it does not react easily.

The noble gases show how a group on the periodic table can share a strong family resemblance.

## Atomic Mass

Many periodic tables show a number called **atomic mass**.

Atomic mass tells the average mass of atoms of an element compared with a standard.

Most of an atom's mass comes from protons and neutrons in the nucleus.

A **neutron** is a particle in the nucleus with no electric charge.

For younger students, it is enough to remember this:

Atomic number tells the number of protons.

Atomic mass gives information about how heavy the atoms are.

Atomic number identifies the element more directly than atomic mass does.

## Isotopes and Average Mass

Atoms of the same element always have the same number of protons, but they can have different numbers of neutrons.

These different forms are called **isotopes**.

Carbon-12 and carbon-14 are both carbon because they both have 6 protons.

But carbon-14 has more neutrons, so it has more mass.

The atomic mass on the periodic table is often a decimal because it is an average of the naturally occurring isotopes of that element.

You do not need to master isotope math yet.

The important idea is that proton number identifies the element, while neutron number can vary.

## Electrons and Chemical Behavior

Electrons are tiny negatively charged particles around the nucleus.

Atoms can gain, lose, or share electrons when they form chemical bonds.

The electrons farthest from the nucleus are often called outer electrons or valence electrons.

These outer electrons help determine how an element reacts.

Elements in the same group often have the same number of outer electrons.

That is why they behave like chemical relatives.

Lithium, sodium, and potassium are all in Group 1 and are all reactive metals.

Fluorine, chlorine, and bromine are all in Group 17 and are all reactive nonmetals.

Helium, neon, and argon are all noble gases and are usually unreactive.

## Why the Table Has an Odd Shape

The periodic table is not a perfect rectangle.

It has gaps, blocks, and two rows often placed underneath the main table.

That shape comes from how electrons are arranged in atoms.

The two rows at the bottom are usually the **lanthanides** and **actinides**.

They are placed below the table to keep the chart from becoming too wide for a page or classroom wall.

They really belong in the larger order of atomic numbers.

The shape may look strange at first, but it is built around real patterns inside atoms.

## The Periodic Table Predicts Behavior

One of the best things about the periodic table is that it helps scientists make predictions.

If you know an element's group, you can often make a good guess about how it behaves.

If an unknown element is in the noble gas group, you would expect it to be mostly unreactive.

If an element is in the alkali metal group, you would expect it to be a reactive metal.

If an element is near silicon in the metalloid region, you might expect it to have mixed properties.

The table does not tell every detail, but it gives powerful clues.

## Patterns Across a Period

As you move across a period from left to right, the elements usually change from metals to metalloids to nonmetals.

For example, Period 3 begins with sodium and magnesium, which are metals.

It moves through aluminum and silicon.

It ends with phosphorus, sulfur, chlorine, and argon, which are nonmetals or noble gas.

This left-to-right change is one reason the table is arranged in rows.

It lets you see a gradual shift in properties.

## Patterns Down a Group

As you move down a group, elements often keep similar chemical behavior, but their atoms get larger and heavier.

Lithium, sodium, and potassium are all alkali metals, but potassium is generally more reactive than lithium.

Fluorine, chlorine, bromine, and iodine are all halogens, but they have different states at room temperature.

Fluorine and chlorine are gases.

Bromine is a liquid.

Iodine is a solid.

Same family, different details.

That is a good way to think about groups.

## Elements, Compounds, and the Table

The periodic table lists elements, not compounds.

Water is not on the periodic table because water is a compound made of hydrogen and oxygen.

Salt is not on the periodic table because sodium chloride is a compound made of sodium and chlorine.

Carbon dioxide is not on the periodic table because it is a compound made of carbon and oxygen.

But the elements inside those compounds are on the table.

The table gives chemists the alphabet of matter. Compounds are like words built from that alphabet.

## A Few Important Elements

Hydrogen is the lightest element and the most common element in the universe.

Helium is used in balloons and is also found in stars.

Carbon is central to living things, fuels, plastics, and diamonds.

Nitrogen makes up most of the air and is important in proteins.

Oxygen is needed for respiration and combustion.

Sodium and chlorine form table salt when chemically joined.

Iron is used in steel and is important in blood.

Copper is used in electrical wiring.

Silicon is used in computer chips and glass.

Uranium is used as fuel in some nuclear power plants.

The periodic table connects tiny atoms to enormous parts of everyday life.

## Do You Have to Memorize the Whole Table?

No.

Scientists do not use the periodic table because they have memorized every box.

They use it because it organizes information so well.

A student should learn how to read the table, find atomic numbers and symbols, recognize groups and periods, and notice major regions such as metals, nonmetals, metalloids, halogens, and noble gases.

Memorizing a few common elements is useful.

Understanding the pattern is far more powerful.

## The Periodic Table in Real Life

The periodic table is not just for laboratories.

It is in your bicycle, your breakfast, your bones, your phone, and the air you breathe.

The calcium in your bones is on the table.

The oxygen in your lungs is on the table.

The carbon in your muscles is on the table.

The silicon in computer chips is on the table.

The iron in steel tools is on the table.

The copper in wires is on the table.

The helium in some balloons is on the table.

The periodic table is a chart of the stuff the world is built from.

## Why Scientists Keep Updating It

The periodic table includes naturally occurring elements and elements made by scientists.

Some very heavy elements are created in laboratories.

They may exist for only a tiny fraction of a second before breaking apart.

Scientists still study them because they help us understand atoms, nuclei, and the limits of matter.

The table is a living scientific tool.

It has a history, a present, and perhaps a future with new discoveries.

## Safety and the Periodic Table

Seeing an element on the periodic table does not mean it is safe to touch, breathe, taste, or mix.

Some elements are harmless in common forms.

Some are poisonous.

Some are radioactive.

Some react violently with water or air.

Some are safe in compounds but dangerous as pure elements.

Never taste unknown substances.

Never smell gases directly.

Never handle element samples unless a teacher says they are safe.

Never assume that a household substance is safe just because it contains a familiar element.

Chemistry rewards curiosity, but it also requires respect.

## Common Misconceptions

One mistake is thinking the periodic table lists every substance.

It does not. It lists elements.

Another mistake is thinking elements in the same row are the most similar.

Often, elements in the same column are more similar because they have similar outer electrons.

A third mistake is thinking atomic mass is the same as atomic number.

Atomic number counts protons. Atomic mass describes average atomic mass.

A fourth mistake is thinking a pure element and a compound containing that element must have the same properties.

Pure sodium metal reacts strongly with water, but sodium in table salt is part of a stable compound.

## How to Use the Periodic Table Like a Scientist

When you look at an element, ask:

- What is its atomic number?
- What is its chemical symbol?
- Is it a metal, nonmetal, or metalloid?
- What group is it in?
- What period is it in?
- Are nearby elements similar?
- Does its group suggest anything about its reactivity?
- Is it commonly found alone, in compounds, or in mixtures?

These questions turn a chart of boxes into a tool for thinking.

## The Big Idea

The periodic table organizes the elements by atomic number and by repeating patterns in their properties.

Each element has a unique atomic number based on its number of protons. The table's rows are periods, and its columns are groups or families. Elements in the same group often behave similarly because their atoms have similar outer electron arrangements. Metals, nonmetals, metalloids, halogens, noble gases, and other families can be found by their positions on the table.

If you remember only one sentence, remember this:

**The periodic table is a map of the elements that reveals patterns in the building blocks of matter.**
