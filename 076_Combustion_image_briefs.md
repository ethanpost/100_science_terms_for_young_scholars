# Image briefs — 076 Combustion

Use when creating `076_Combustion_02.png` through `076_Combustion_07.png`. Each file is referenced in `076_Combustion.md` at the placement noted below.

`076_Combustion_01.png` already exists at the chapter top (fire triangle). Brief below for consistency if it is ever redrawn.

**Style** (from `_create_more_images.md`): crude, funny, hand-drawn explainer cartoon; stick-figure characters; rough black outlines; mostly white background; selective flat accent colors. Labels, arrows, exaggerated faces, simple metaphors. Minimalist, humorous, concept-first, intentionally rough. Color sparingly: **red** = danger, **yellow** = warning, **blue** = oxygen/air/systems, **orange** = heat/flame, **gray** = engines/machinery. Vary panel width/height — not every image the same aspect ratio. Ages 11–13; appeal to campfires, dirt bikes, grills, rockets without glorifying unsafe fire play.
